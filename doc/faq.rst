.. _faq:
===========================
Frequently Asked Questions
===========================

.. currentmodule:: IMP.bff

Here we try to give some answers to questions that regularly pop up.

What is the project name (a lot of people get it wrong)?
--------------------------------------------------------
IMP.bff that stands for the Bayesian Fluorescence
Framework (BFF) of the Integrative Modeling Platform (IMP).

How can I get help?
-------------------------------------------------
For help either raise a GitHub issue `issue <https://github.com/fluorescence-tools/imp.bff/issues>`_.
